# 🏛️ Otara CMS: ადმინისტრირების სრული სახელმძღვანელო

მოგესალმებით თქვენი საიტის მართვის პანელში. ეს დოკუმენტი დაგეხმარებათ დეტალურად გაერკვიოთ, თუ როგორ მართოთ საიტის თითოეული ელემენტი პროგრამისტის დახმარების გარეშე.

## 1. სწრაფი ნავიგაცია (ზედა ზოლი)
ადმინ პანელის ზედა ნაწილში დაგხვდებათ ორი მნიშვნელოვანი ღილაკი:
*   **⚙️ საიტის პარამეტრები (წითელი):** ეს არის ყველაზე მნიშვნელოვანი ღილაკი. მასზე დაჭერით პირდაპირ გადახვალთ საიტის მთავარ კონფიგურაციაში (სახელი, ტელეფონი, ლოგო, ფონური სურათები).
*   **👁️ საიტის ნახვა (ლურჯი):** ხსნის საიტის მთავარ გვერდს ახალ ფანჯარაში, რათა სწრაფად გადაამოწმოთ თქვენი შეტანილი ცვლილებები.

## 2. საიტის გლობალური პარამეტრები (Global Site Settings)
ეს სექცია საიტის "ტვინია". აქ შეტანილი ცვლილებები აისახება ყველა გვერდზე.

### 🏷️ ბრენდინგი და კონტაქტი
*   **საიტის სახელი:** თქვენი ბრენდის სახელი (ჩანს ლოგოს ნაცვლად და ფუტერში).
*   **საკონტაქტო ინფორმაცია:** ელ-ფოსტა, ფიზიკური მისამართი და WhatsApp ნომერი.
*   **WhatsApp ნომერი:** ჩაწერეთ საერთაშორისო ფორმატში (მაგ: +995 5xx xxx xxx). ამ ნომერზე მოვა კლიენტების შეტყობინებები.

### 🖼️ Hero სექცია (მთავარი ბანერი)
*   **მთავარი ფოტო:** სურათი, რომელიც ჩანს საიტის გახსნისთანავე. რეკომენდებული ზომა: 1920x1080 პიქსელი.
*   **მთავარი სათაური და ქვესათაური:** ტექსტი, რომელიც გადაეფარება სურათს.

### 📄 "ჩვენს შესახებ" (About Page)
*   აქ შეგიძლიათ მართოთ About გვერდის მთლიანი შინაარსი, მათ შორის თქვენი **ბიზნეს ფილოსოფია**.
*   ტექსტების დასაწერად გამოიყენება ვიზუალური ედიტორი (TinyMCE), სადაც შეგიძლიათ ტექსტი გახადოთ სქელი (**Bold**), დაამატოთ სიები ან ლინკები.

### 🤖 Telegram შეტყობინებები
*   აქ იწერება ბოტის ტოკენი და ჩატის ID. ამის მეშვეობით საიტი მომენტალურად გატყობინებთ ახალი ჯავშნის ან კითხვის შესახებ პირდაპირ ტელეგრამში.

## 3. ტურების და ექსპედიციების მართვა
ეს არის თქვენი ბიზნესის გული. სექცია შედგება სამი ნაწილისგან:

### 🗺️ ტურები და ექსპედიციები
ახალი ტურის დამატებისას მიუთითეთ:
*   **სათაური და აღწერა:** დეტალური ინფორმაცია ტურის შესახებ.
*   **ფასი და ხანგრძლივობა:** ციფრების სახით.
*   **მთავარი სურათი:** რომელიც გამოჩნდება ტურის ბარათზე.
*   **Is Featured:** თუ მონიშნავთ, ეს ტური გამოჩნდება მთავარ გვერდზე "რჩეული ტურების" სექციაში.

### 📍 რეგიონები და დესტინაციები
სანამ ტურს დაამატებთ, უნდა შექმნათ ლოკაცია (მაგ: სვანეთი, კახეთი).
*   თუ მონიშნავთ **"Is Top Destination"**, ეს რეგიონი გამოჩნდება მთავარი გვერდის "Top Destinations" სლაიდერში.

### 📁 ტურის კატეგორიები
აქ მართავთ ტურის ტიპებს (მაგ: Trekking, Wine Tour). თითოეულ კატეგორიას შეგიძლიათ აურჩიოთ შესაბამისი იკონკა სიიდან.
